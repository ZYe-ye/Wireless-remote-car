比赛记录
1、无线遥控采用的是nrf24l01（2.4G无线模块），比赛过程中发生串频(这也是我第一次遇到这个问题)，以后比赛记得将频道再做修改
2、小车主控板采用的是STM32F103C8T6,用到了PC13~PC15的gpio，在测试的时候，如果出现模块烧了之后，核心板的PC13~PC15就会出现问题，所以以后尽可能不要使用这几个gpio，因为这几个gpio比较特殊，其他gpio没出现这个问题
3、小车原定计划是使用单电源供电，结果在测试过程中，由于这次要驱动6个舵机和4个编码电机，所以单电源无法带动，并且出现模块烧了以和莫名熄火的情况，解决方法：后面改成双电源，一个电源给主控板和编码电机，另一个给6个舵机；因为之前买的电池有问题，所以换成了航模电池，解决了莫名熄火的问题。
